# Stock Portfolio Monitor App

A web-based dashboard to track US and HK stocks, performance metrics, and buy/sell signals.

## Features
- Portfolio Overview (Total Value, P&L, ROI %)
- Real-time Stock Prices (US + HK)
- Technical Indicators (RSI, MACD)
- Buy/Sell Signals
- Weekly Performance Summary
- Price Alerts

## Tech Stack
- Python 3.10+
- Streamlit (Frontend)
- yfinance (Data)
- pandas-ta (Technical Analysis)

## Installation

```bash
pip install -r requirements.txt
```

## Run Locally

```bash
streamlit run app.py
```

## Deploy to Streamlit Cloud
1. Push to GitHub
2. Go to share.streamlit.io
3. Connect your GitHub repo
4. Deploy!
